## 0. 프로젝트 개요

✔ 프로젝트명 : GGONG(꽁수래 꽁수거)

✔ 한줄 소개 : Web IoT를 기반으로 한 투표식 담배꽁초 수거함

✔ 개발 기간 : 23.07.10 ~ 23.08.18 (6주)

✔ 팀원 : 정예지 김동현 민병기 전주영 정의석 홍성민

✔ 사용 기술스택 : SpringBoot, React, Docker, AWS + (4. 시스템 아키텍처 참고)
